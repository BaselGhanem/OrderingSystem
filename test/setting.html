إدارة وتحديث الأصناف

تحديث شامل عبر Excel أو تعديل وحذف مباشر من جدول المنتجات الحالي.

أسرع طريقة لتعديل الأصناف: نزّل ملف الأصناف كما هو من Firebase، عدّل name أو code أو productCode أو price، ثم ارفع نفس الملف. النظام يحدث السجلات حسب Document_ID ولا يحذف أي صنف غير موجود في الملف. إذا تغيّر كود الصنف، سيتم تبديل الكود القديم بالجديد داخل الطلبيات الموجودة فقط، بدون تعديل السعر أو الكمية أو الإجمالي.
1
تنزيل ملف الأصناف الحالي من قاعدة البيانات.
2
تعديل الاسم أو الكود أو السعر داخل Excel فقط.
3
رفع نفس الملف لتحديث Firebase مباشرة. تغيير الكود ينعكس على كودات الطلبيات القديمة فقط.
